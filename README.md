# NexusMind AI

NexusMind AI is an Adaptive & Trustworthy Generative AI system built using Retrieval-Augmented Generation (RAG).

## Features

- PDF Upload
- Intelligent Text Chunking
- ChromaDB Vector Database
- Semantic Search
- Llama 3 Integration (Ollama)
- Source Citation (Page & Chunk)
- Confidence Score
- FastAPI Backend

## Tech Stack

- Python
- FastAPI
- ChromaDB
- Sentence Transformers
- Ollama
- Llama 3
- LangChain
- PyPDF
